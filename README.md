# 🧪 VR Molecular Chemistry Lab

## 📌 Overview
This project is an interactive **VR Molecular Chemistry Lab** built using Unity and XR technologies.

It allows users to grab atoms, combine them, and form valid molecules with real-time feedback. The system detects correct molecular structures and visualizes them with proper bonding representation.

---

## 🎯 Purpose of the Project
The main purpose of this project is to:

- Demonstrate **interactive learning of molecular chemistry in VR**
- Showcase **XR interaction design and implementation**
- Focus on **core mechanics and system architecture**
- Build a system for **real-time molecule formation and validation**

This project prioritizes **functionality and logic** over visual complexity.

## 🧩 Features

- Atom grabbing using XR controllers  
- Real-time bond detection system  
- Molecule formation and validation  
- Visual feedback for valid molecules  
- Molecule listing UI panel  
- Reset and reuse system  

---

## 🛠️ Tools & Technologies Used

- Unity (Game Engine)
- XR Interaction Toolkit
- OpenXR Plugin
- C# (Scripting)

### Additional Notes:
- Atoms and molecules are created using **Unity built-in primitives**
- UI, particle effects, and audio use **external packages**
- Focus was placed on **mechanics and architecture rather than custom 3D assets**

## 🧠 AI Usage

ChatGPT was used as a support tool for:

- Assisting in code implementation

All core logic, interaction flow, and system design were independently developed.

## ⚙️ Setup Instructions

1. Clone the repository  
2. Open the project in Unity (latest stable version recommended)  
3. Enable XR Plugin Management and OpenXR  
4. Build for Android (Meta Quest compatible)
